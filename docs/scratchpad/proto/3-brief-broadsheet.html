§ 01

Built for three readers

One artifact serves the person deciding, the agent acting, and the wiki that keeps the record.

For humans

A page you read, mark up, and decide inside, with editorial typography that makes long-form pleasant.

For agents

Structured packets read straight back out, so the next agent picks up exactly where the last one left off.

For your wiki

One file to drop anywhere, with a plain Markdown twin so the record stays editable and durable.

Tip

Keep everything in one file. Run dossier build report.json and the page, the Markdown, and the agent digest all ship together, no network required.
